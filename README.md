# Climalert

Climalert es un sistema de monitoreo climático que consulta periódicamente un proveedor meteorológico externo para obtener información del clima en tiempo real, analiza las condiciones registradas y envía alertas automáticas por correo electrónico cuando se cumplen determinados criterios.

## Tecnologías utilizadas

- Java 21
- Spring Boot 4
- Maven
- WeatherAPI
